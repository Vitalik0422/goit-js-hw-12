# Image Search Gallery

A web application for searching and browsing images by keyword, built with Vite.

## Features

- Search images by keyword via Pixabay API
- Smooth gallery reveal after all images are loaded
- Full-size image preview in a modal window
- Animated loading indicator
- Toast notifications for empty results or errors

## Technologies

- Vite — project bundler
- Axios — HTTP requests
- SimpleLightbox — modal image viewer
- iziToast — notifications
- css-loader — loading indicator

## How to Use

1. Type a keyword in the search field
2. Press Search
3. Browse results in the gallery
4. Click any image to open a full-size preview
